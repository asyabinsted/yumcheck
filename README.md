## YumCheck

YumCheck is a mobile app that helps shoppers scan groceries, review ingredient details, and keep their pantry organized with up-to-date product insights.

## Current Features
- Instant barcode scanning with automatic product lookup
- Ingredient analysis powered by reverse image search for label photos
- Product suggestions to surface healthier or preferred alternatives
- Quick actions to add, remove, or clean up items in the local database
- Live connection to the Open Beauty Facts database for baseline product data
- Supabase-backed local product database for custom entries and syncing
- Supabase ingredient rating database seeded from the official Cosing dataset
- Ready-to-run SQL scripts for managing row-level security policies
